# Changelog

## v1.1.0 - 2026-03-22

### Added
- 新增 ❤魅力模式（左红右紫分屏补光）
- 新增魅力模式红紫分隔宽度调节
- 新增底部设置收纳开关，默认收起设置面板
- 新增拍照后“查看照片”入口
- 新增赞助说明与赞赏码展示

### Changed
- 普通模式色调逻辑改为：左冷白 / 中粉润 / 右红润
- 色调滑杆默认置中（粉润）
- 预览框支持拖动与按钮切换大小
- 补光引导文案与实时数值显示优化
- 照片保存路径改为系统图库 `Pictures/GlowSelfie`

### Fixed
- 修复 CI 中 AGP 配置兼容问题
- 修复设置面板展开时与底部按钮栏重叠问题
- 修复预览比例计算导致的拉伸问题
