# 快速安装指南

第一步：上传到 GitHub

1. 访问 https://github.com/new
2. 创建新仓库 GlowSelfie
3. 点击 upload files
4. 把整个 GlowSelfie 文件夹拖进去
5. 点击 Commit changes

第二步：自动构建 APK

1. 点击仓库的 Actions 标签
2. 等待构建完成（绿色勾）
3. 点击 Build APK
4. 下载 GlowSelfie-APK.zip

第三步：安装到手机

1. 解压得到 app-debug.apk
2. 传到 Realme GT NEO5
3. 点击安装
4. 允许未知来源
5. 完成
