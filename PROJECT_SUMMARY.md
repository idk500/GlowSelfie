# GlowSelfie 项目说明

## 项目名称
GlowSelfie - 柔光自拍

## 功能特性

1. 粉色补光：屏幕自动以最高亮度显示粉色，作为自拍补光灯
2. 小窗预览：屏幕中央显示摄像头实时画面，方便摆姿势
3. 调色盘：两个滑动条，分别调节亮度（强度）和粉色色调
4. 一键拍照：点击拍照按钮，自动保存到相册
5. 简单操作：打开即用，无需学习

## 技术规格

应用ID：com.glowselfie.app
版本：1.1.0
最低系统：Android 8.0 (API 26)
目标系统：Android 14 (API 34)
开发语言：Java
相机实现：Android Camera（前置优先）

## 使用方法

1. 打开应用，允许相机权限
2. 默认进入普通补光模式（左冷白 / 中粉润 / 右红润）
3. 可展开设置，调节补光强度与色调
4. 可切换 ❤魅力模式（左红右紫分屏补光）
5. 看小窗预览调整姿势并拍照
6. 照片保存到系统图库（Pictures/GlowSelfie）
