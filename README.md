# GlowSelfie

GlowSelfie 是一款面向暗光自拍场景的屏幕补光相机应用。它通过高亮屏幕 + 小窗预览，帮助你在夜晚或室内弱光环境快速出片。

## 核心功能

- 普通补光模式：色调滑杆支持左冷白 / 中粉润 / 右红润
- ❤魅力模式：屏幕左红右紫分屏补光，支持红紫分隔宽度调节
- 预览框可拖动，支持一键切换预览尺寸
- 默认最大屏幕亮度补光
- 拍照后可直接查看图片
- 照片保存到系统图库（Pictures/GlowSelfie）

## 快速安装

1. 打开 GitHub Releases 页面下载最新 APK
2. 发送到手机并安装
3. 首次启动允许相机权限

详细可见 [INSTALL.md](INSTALL.md)。

## 使用建议

1. 默认先用普通模式，强度建议 70% 以上
2. 色调中间位置为粉润；左侧偏冷白显白；右侧偏红润显气色
3. 开启 ❤魅力模式 时建议横屏，分屏补光更明显
4. 预览窗拖到前摄附近，构图更顺手
